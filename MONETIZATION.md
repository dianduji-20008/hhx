# 槐花香（HHX）— 变现方向研究清单

> 目标：赚钱。不设品牌限制，什么赚钱做什么，每个方向都要研究验证。

## 方向一：数字产品（模板/教程）

### 1.1 Notion/飞书模板
- **投入**：几乎为零（AI 辅助批量生产）
- **定价**：¥9.9-49/套
- **渠道**：小红书、即刻、闲鱼、淘宝
- **变现速度**：最快，今天做明天卖
- **待研究**：
  - [ ] Notion 模板市场现状，哪些品类卖得好
  - [ ] 飞书模板是否有官方市场
  - [ ] 小红书上模板类内容的流量和转化数据
  - [ ] 竞品分析：谁在卖模板，定价多少，月销量多少
  - [ ] 模板制作工具链（Notion API / 飞书多维表格）
  - [ ] 交付方式（链接分享 / 文件导出）

### 1.2 课程/教程
- **投入**：中等（内容创作时间）
- **定价**：¥49-199
- **渠道**：小报童、知识星球、B站、小红书
- **卖点**：「AI 编程实战：一个人如何做出上线产品」— 你正在做的事就是内容
- **变现速度**：快，有真实经验就能卖
- **待研究**：
  - [ ] AI 编程教程的市场需求（搜索量、竞品课程）
  - [ ] 小报童/知识星球的开门槛和分成比例
  - [ ] 同类课程定价和销量（如：AI 编程、独立开发、一人公司）
  - [ ] 内容形式：文字专栏 vs 视频课 vs 直播
  - [ ] 最小化 MVP：先出一篇免费内容测试反响

## 方向二：工具站广告（Toolbox 现有方向）

- **投入**：低（已有基础）
- **变现**：AdSense + 国内广告联盟
- **回报周期**：6-12 月
- **天花板**：月入几千
- **待研究**：
  - [ ] AdSense 审批流程和国内工具站的实际 CPM
  - [ ] 国内广告联盟（百度联盟、搜狗联盟）的门槛和收益对比
  - [ ] PDF 工具站竞品的流量和收入数据
  - [ ] 工具站 SEO 关键词搜索量排名（PDF > 图片 > JSON > 其他）
  - [ ] 日 PV 1000 的工具站实际月收入

## 方向三：SaaS 订阅（生意本 BizBook）

- **投入**：中（4-6 周开发）
- **变现**：¥9.9-29/月订阅
- **回报周期**：3-6 月
- **天花板**：月入几万（需要规模化获客）
- **待研究**：
  - [ ] 国内小商家 SaaS 付费意愿的真实数据
  - [ ] 报价工具竞品深度分析（功能、定价、用户评价）
  - [ ] 个体户注册流程和微信支付接入周期
  - [ ] 获客渠道成本：微信群 / 抖音 / 小红书 / SEO
  - [ ] ¥9.9/月定价是否太低，能否支撑运营
  - [ ] 先做 H5 还是直接做小程序的 A/B 决策

## 方向四：Chrome 扩展

- **投入**：低（1-2 周开发）
- **变现**：免费+付费功能 / 订阅
- **回报周期**：1-2 月
- **天花板**：中等
- **待研究**：
  - [ ] Chrome Web Store 付费扩展的销量数据
  - [ ] 哪些品类扩展卖得好（效率工具 / 开发者工具 / 社交增强）
  - [ ] 竞品分析：排名靠前的付费扩展
  - [ ] 扩展分发和获客方式（Store SEO / 内容营销）
  - [ ] 技术方案：Manifest V3 限制和付费实现
  - [ ] 具体产品方向：网页截图标注 / 批量下载 / 去广告 / 标签管理

## 方向五：微信机器人/插件

- **投入**：中
- **变现**：按月收费
- **回报周期**：1-3 月
- **天花板**：中等
- **待研究**：
  - [ ] 微信机器人技术方案（企微 API / itchat / wechaty）
  - [ ] 合规风险（微信对第三方客户端的态度）
  - [ ] 企业微信 vs 个人微信的市场差异
  - [ ] 群管理机器人的竞品和定价
  - [ ] 报价机器人（和生意本结合）的可行性
  - [ ] 微信小程序插件市场

## 方向六：内容/自媒体

- **投入**：中（持续输出）
- **变现**：广告 / 带货 / 知识付费
- **回报周期**：3-6 月
- **天花板**：高
- **待研究**：
  - [ ] 独立开发者自媒体的变现案例
  - [ ] B站 / 小红书 / 抖音的内容形式和算法偏好
  - [ ] 「一个人做产品」类内容的流量数据
  - [ ] 自媒体引流到自有产品的转化率
  - [ ] 最小化起步：哪个平台最适合技术型创作者

## 研究原则

1. **每个方向花 1-2 天研究**，不急着动手
2. **看真实数据**，不看别人吹的
3. **找到至少 1 个成功案例**（有人靠这个赚到了钱）
4. **算清投入产出比**：投入多少时间，预期多少收入
5. **选 1-2 个方向并行**，不贪多
6. **先赚到第一块钱**，再优化

## 决策标准

一个方向值得做，当且仅当：
- 有人靠类似产品赚到了钱（验证需求）
- AI 辅助下 1 人可在 2 周内出 MVP（可控投入）
- 有明确的获客渠道（不是「做了再说」）
- 第一个月有概率赚到钱（不是等 6 个月）
